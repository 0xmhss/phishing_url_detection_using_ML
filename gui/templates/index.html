Project

SafeLink was developed as part of our Cybersecurity coursework to demonstrate how machine learning can be applied to detect phishing URLs in real time. The project combines URL feature engineering, a trained classification model, and a web-based interface so the detector can be tested interactively rather than only from the command line.
